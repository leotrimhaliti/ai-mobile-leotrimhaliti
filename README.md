# University Bus Tracker MVP

A simple MVP web app inspired by the **Trafiku Urban** app, designed to track **university buses in real-time**. Built with **React Native** for the frontend and **Supabase** for the backend.

---

## Features

- **Live bus tracking:** See where buses are in real-time on a map.  
- **Basic user interface:** Minimal design to view buses and their status.  
- **Supabase backend:** Stores bus and location data for easy retrieval.  
- **Map integration:** Google Maps and Apple maps   

---

## Tech Stack

- **Frontend:** React Native + Expo  
- **Backend:** Supabase (PostgreSQL + Realtime)  
- **Maps:** react-native-maps & google maps api
